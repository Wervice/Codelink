Using

To set up a new link, click the button (e.g.: Terminal) and enter a command that runs the application.
You can select one from the list bellow.
Please not that some applications have to be in PATH and installed to work.
The confirm and the application will launch. The link is now set up.
If you want to reset it, right-click it.
