Codelink Manual

Codelink features multiple tools, you can use and play with.

Links

The first one is the ability to link your programs with Codelink.
You can simply click one of the 3 first icons on the main windows.
There you can setup a command to run the app and it will start.
This then is stored for the next time you click it. If you want to reset it, right click it.

FileTree

FileTree is a full file manager for developers. It allows you to easily navigate with your keyboard, run special commands very quick and more.
You can run it by clicking the 4th icon on the main screen.

Browser

Codelink's browser is a Chromium based browser with a bug fixing assistant and more features. It has no history.
You can run it, by clicking the mug icon on the main screen.
